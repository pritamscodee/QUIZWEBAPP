# Quiz Web App

A full-stack quiz application with role-based access (Admin/Student). Built with React (Vite), Node.js (Express), MongoDB Atlas, and deployed on Vercel + Render(https://quizwebappfrontend-5rtdamiay.vercel.app/).

## Features

- User authentication (JWT)
- Role-based dashboards (Admin can create quizzes; Students can attempt)
- Quiz creation with dynamic questions
- Timed quiz taking with auto-submit
- Result tracking and leaderboard
- Neobrutalism UI design

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Express, MongoDB Atlas
- **Deployment**: Vercel (frontend), Render (backend)
